Project Lovelace

A computer game about computing

Society Has Collapsed

Following a devastating solar flare, Project Lovelace was bornâ€”a global initiative to rebuild computing from the ground up, named after the first programmer Ada Lovelace. You are tasked with recreating computing from the ground up, starting from basic logic, just as she did.

Based on the Bevy game engine

Bevy is an early stages game engine in rust, but its powerful ECS (Entity Component System) model and excellent integration with rust language features make it more fun than any other game engine i've worked in before.
